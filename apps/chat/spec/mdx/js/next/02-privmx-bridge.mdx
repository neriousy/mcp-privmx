  <div className="fd-step">**Clone the PrivMX Bridge Docker repository**</div>  

  <Tabs items={["HTTPS","SSH", "GitHub CLI"]}>
    <Tab>
      ```sh
      git clone https://github.com/simplito/privmx-bridge-docker.git
      ```
    </Tab>

    <Tab>
        ```sh
      git clone git@github.com:simplito/privmx-bridge-docker.git
      ```
    </Tab>

    <Tab>
        ```sh
      gh repo clone simplito/privmx-bridge-docker
      ```
    </Tab>
  </Tabs>

  <div className="fd-step">**Spin up the PrivMX Bridge instance**</div>

  Navigate into the cloned repository and run the setup script:

  ```sh
  cd privmx-bridge-docker
  ./setup.sh
  ```
  This script will:

  - Start the necessary Docker containers
  - Generate an **API Key**
  - Create a default **Solution** and **Context**
